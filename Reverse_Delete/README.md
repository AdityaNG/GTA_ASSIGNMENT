# Reverse Delete to find MST

# Getting Started

Compilation on Linux:
```
make
```
Usage:
```
./ReverseDelete
```

Sample output:
```
Edges in MST
(3, 4) 
(0, 7) 
(2, 3) 
(2, 5) 
(0, 1) 
(5, 6) 
(2, 8) 
(6, 7) 
Total weight of MST is 37
```

## Compiling and Running:

On Mac and Linux:
```bash
g++ -std=c++17 main.cpp -o Boruvka
./ReverseDelete < ../input.txt > output.txt
```

On Windows:
```bash
g++ -std=c++17 main.cpp -o Boruvka
ReverseDelete.exe < ../input.txt > output.txt
```
